# 🤖 Bio-Inspired AI Robot: Pure Perception & Attention Tension Vectors

An advanced, light-weight cognitive architecture simulating robotic consciousness. This project demonstrates how an autonomous agent can achieve system equilibrium through **Pure Perception** (avoiding computational overthinking), combined with a mathematical **Attention Tension Vector** model and a **Dual-Layer Auditory System**.

## 🧠 Architecture Pillars

1. **Pure Perception (Kinematic Core)**: Connects sensor inputs directly to behavioral responses (Speed vs. Distance) to trigger instant braking, bypassing heavy legacy log parsing.
2. **Attention Tension Vectors**: Dynamically allocates the robot's spatial focus using Euclidean distance and environmental importance weights.
3. **Dual-Layer Auditory Loop**: Keeps the robot in a low-energy "Zen State" (Passive Hearing) until a decibel threshold or specific NLP "Keyword" triggers full cognitive processing (Active Listening).

## 🚀 Getting Started

### Prerequisites
Make sure you have Python 3.x installed along with the required libraries:
```bash
pip install numpy pandas
```

### Running the Simulation
Execute the core script to see the telemetry data across 4 distinct environmental scenarios:
```bash
python main.py
```

* **Complexity**: Vectorized focus calculation operates at $O(N)$ efficiency, allowing near-instant priority switching.
